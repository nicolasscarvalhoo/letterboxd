# Letterboxd Clone

Este projeto é uma réplica da interface do **Letterboxd**, desenvolvida com **React Native** e **Expo**. O objetivo é recriar a experiência de navegação social de filmes com um design moderno e fluido.

##  Tecnologias Utilizadas

- **React Native**: Framework para construção da interface mobile.
- **Expo**: Ferramenta para simplificar o fluxo de desenvolvimento.
- **React Navigation**: Gerenciamento de navegação (Bottom Tabs).
- **TypeScript**: Tipagem estática para maior segurança no código.
- **Ionicons**: Biblioteca de ícones integrada.

##  Funcionalidades

- [x] **Navegação em Abas**: Estrutura de feed e perfil.
- [x] **Feed de Filmes**: Listagem de filmes populares e atividade de amigos.
- [x] **Perfil do Usuário**: Exibição de favoritos e histórico de atividade.
- [x] **Design Responsivo**: Interface adaptada para mobile e web.

##  Como executar o projeto

1. **Clone o repositório:**
   ```bash
   git clone <URL-DO-SEU-REPOSITORIO>
   cd letterboxd
2. **Instale as dependências:**
   ```bash
   npm install
3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm expo start 
4. **Visualização:**
   - Pressione 'a' para abrir no emulador Android.
   - Pressione 'i' para abrir no emulador iOS.
   - Pressione 'w' para abrir no navegador.
